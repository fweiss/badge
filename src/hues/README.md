# Hues

This is a simple FastLED sketch that cycles all 64 LEDs through 256 hues.

## Current profiles

hardware: CJMCU-8x8,
device: Arduino Pro Mini,
Power: 5 VDC 2 A wall wart

| Parameter | Value | Units |
|-----------|------:|-------|
| hue       | 0-255 |       |
| sat       | 255   |       |
| val       | 120   |       |
| delay     | 15    | ms    |
| Vdd       | 5.22  | V     |
| Imax      | 272   | mA    |
| Imin      | 193   | mA    |
| Iavg      | 221   | mA    |
